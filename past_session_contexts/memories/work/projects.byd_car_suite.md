# work.projects.byd_car_suite
Last updated: 2026-03-26

The user built an Android app suite called byd-car-suite for their BYD Destroyer 05 (2025 DM-i) car using Kotlin, Jetpack Compose, Hilt, and Room. The suite includes an OBD-II diagnostics app, a trip/fuel tracker, a quick controls panel, and a HUD dashboard overlay.
